# DataLog_Backend

# General installation
- Open the project in Visual Studio Code
- Command line run: 
- pipenv shell 
- pipenv install -r requirement.txt
- python main.py

# About the app
This is the backend of DataLog web application.

# Build with
- MongoDB
- FastAPI
- Python
